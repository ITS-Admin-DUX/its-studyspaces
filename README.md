## Components

  * [Github Pages](https://pages.github.com/) [more info](#github-pages)

## Getting Started
  
### AstroJs
#### Local Development
```sh
npm install
npm run dev
```

#### Build
```sh
npm run build
```

### Github Pages
#### Actions
The site is built and deployed using Github Actions. The workflow is defined in `.github/workflows/main.yml`.
